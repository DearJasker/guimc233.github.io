B站相关功能

B站相关功能

1.搜素功能

  • /bili-search 番剧 \ # 搜索番剧
  • /bili-search 影视 \ # 搜索影视
  • /bili-search 用户 \ # 搜索用户

2.查询功能 (以下功能都有自动匹配链接,可直接发送链接)

  • /bili-info aid \ # 查询视频 (AV号)
  • /bili-info bvid \ # 查询视频 (BV号)
  • /bili-info dynamic \ # 查询动态
  • /bili-info live \ # 查询直播间

3.订阅 (仅群管理员/群主和超管可用)

  • 视频订阅

    • /bili-video add \ /bili-video 添加 \
    • /bili-video list /bili-video 列表
    • /bili-video stop \ /bili-video 停止 \
  • 动态订阅
    • /bili-dynamic add \ /bili-dynamic 添加 \
    • /bili-dynamic list /bili-dynamic 列表
    • /bili-dynamic stop \ /bili-dynamic 停止 \
  • 直播订阅
    • /bili-live add \ /bili-live 添加 \
    • /bili-live list /bili-live 列表
    • /bili-live stop \ /bili-live 停止 \

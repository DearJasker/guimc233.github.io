B站相关功能

B站相关功能

此为他人插件,并非我编写的

1.搜素功能

  • /bili-search 番剧 [keyword] # 搜索番剧
  • /bili-search 影视 [keyword] # 搜索影视
  • /bili-search 用户 [keyword] # 搜索用户

2.查询功能 (以下功能都有自动匹配链接,可直接发送链接)

  • /bili-info aid[id] # 查询视频 (AV号)
  • /bili-info bvid[id] # 查询视频 (BV号)
  • /bili-info dynamic[id] # 查询动态
  • /bili-info live[id] # 查询直播间

3.订阅 (仅群管理员/群主和超管可用)

  • 视频订阅

    • /bili-video add [uid]
      /bili-video 添加 [uid]
    • /bili-video list
      /bili-video 列表
    • /bili-video stop [uid]
      /bili-video 停止 [uid]
  • 动态订阅
    • /bili-dynamic add [uid]
      /bili-dynamic 添加 [uid]
    • /bili-dynamic list
      /bili-dynamic 列表
    • /bili-dynamic stop [uid]
      /bili-dynamic 停止 [uid]
  • 直播订阅
    • /bili-live add [uid]
      /bili-live 添加 [uid]
    • /bili-live list
      /bili-live 列表
    • /bili-live stop [uid]
      /bili-live 停止 [uid]
